# Flipkart Sales Data Analysis Dashboard (Excel Project)

## 📊 Project Overview

This project is an interactive **Sales Analysis Dashboard created in Microsoft Excel** using Flipkart sales data.
The goal of the project is to analyze sales performance, profit trends, and product performance using different visualization techniques and KPIs.

The dashboard helps in understanding key business insights such as **total sales, total purchases, and total profit**, along with state-wise, month-wise, and product-wise analysis.

## 🚀 Key Features

* **Interactive Category Slicer** to filter data by product category
* **KPI Cards** displaying:

  * Total Sales
  * Total Purchase
  * Total Profit
* **Pie Chart Visualization** showing KPI distribution
* **State-wise Sales Map Chart** for geographical sales analysis
* **Month-wise Sales & Profit Chart (Dual Axis)** to compare sales and profit trends
* **Product-wise Sales Tree Map Chart** to identify top-performing products
* **Profit-wise Donut Chart** for visual profit breakdown

## 📈 Insights Generated

* Identify top-performing states by sales
* Analyze monthly sales and profit trends
* Understand which product categories generate the most revenue
* Evaluate profit distribution across products

The objective of this project is to demonstrate **data analysis and dashboard creation skills using Excel**, and to showcase the ability to convert raw data into meaningful business insights.
